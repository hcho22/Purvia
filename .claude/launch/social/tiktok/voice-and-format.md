# TikTok: voice and format

**Logic of the platform for this launch:** TikTok's dev audience rewards screen-recording authenticity, not production value. The format is a terminal, a real command, and a voiceover that sounds like a code review. The hook is always the uncomfortable number, never the pitch. Videos are storyboarded only (brief §Phase 4: no rendered video in this build); every shot names the exact command or artifact so filming is mechanical.

Format rules:
- 20-60 seconds. Hook lands inside 3 seconds and is a real number or a real admission, on screen and spoken.
- Terminal: dark theme, font large enough for phone screens (24pt+), one command per shot, output pauses long enough to read.
- On-screen text: lowercase, short, no emoji, no exclamation marks. Voiceover: first person, spoken-word versions of the written voice rules; numbers read digit by digit where precision matters ("zero point one one zero").
- Every video ends on where to verify, not a call to action to follow.
- No music over voiceover; captions burned in (accessibility plus sound-off viewing).
- Nothing staged: if a command output is shown, it is the real output of the real command on the real repo. If the repo changes and a number moves, re-film rather than re-caption.

Cadence: one per week; T-01 (the confession) first, T-04 (currently broken) inside the first two weeks for the same reason the site leads with the honest section.
