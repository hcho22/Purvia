Receipt 002 / the math

Post-filtering permissions breaks retrieval. Quietly.

Retrieve top-10, then drop what the viewer cannot see. If they can see 5% of the corpus:

E[visible] = k × selectivity
= 10 × 0.05 = 0.5 chunks

Half a chunk, on average. A multi-hop question needs two. Fetching top-100 does not rescue it: similarity stops ranking the visible set against itself. The fix is the ACL inside the SQL predicate, so the planner only ranks what the viewer may see.

Receipt 003 / the predicate

Permissions live inside the SQL.

Owner-OR-ACL and tenant membership, checked in the retrieval function itself, under the viewer's JWT.

and (c.user_id = auth.uid()
     or ca.chunk_id is not null)
       -- owner OR chunk-level ACL
and exists (
  select 1
  from workspace_membership wm
  where wm.workspace_id
          = d.workspace_id
    and wm.user_id = auth.uid()
       -- tenant boundary, from the JWT
)

No backend-passed tenant id exists to forget. A missing filter can only narrow, never widen.

Receipt 005 / the gate

The bot never decides to escalate. The gates do.

No escalate() tool. Deterministic control flow, from the docstring that ships:

retrieve (hybrid, ONCE)
  → retrieval gate
      → weak   ⇒ escalate now
      → strong → draft
        → faithfulness gate
            → faithful   ⇒ answer
            → unfaithful ⇒ escalate

One buyer-set risk number: a 5% false-resolve ceiling, enforced by the weekly eval gate. A judge timeout escalates. Failure fails closed.

Receipt 007 / the lesson

An alarm that always fires is no alarm.

Before: two-sided
| keyword | +0.030 | ✗ |
| vector  | +0.210 | ✗ |
| hybrid  | +0.210 | ✗ |

Improvements flagged as failures. Red on every snapshot it published, so everyone learns to ignore the column.

After: one-sided (US-118)
| keyword | -0.000 | ✓ |
| vector  | +0.000 | ✓ |
| hybrid  | +0.000 | ✓ |

Only a real drop past tolerance flags. Improvements always pass. The alarm means something again.

This is hardware-validation discipline applied to RAG: the bug cost one migration; the alarm that failed to catch it cost a redesign.
